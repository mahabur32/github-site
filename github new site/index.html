FAQ

সচরাচর জিজ্ঞাসা

এটি সম্পূর্ণ আপনার Personal account। আপনার নিজের GitHub অ্যাকাউন্টেই সব অ্যাক্টিভেট হবে — কারো সাথে শেয়ার করা হয় না।

হ্যাঁ, আগে পেমেন্ট করতে হবে। কারণ এটি ডিজিটাল সার্ভিস — পেমেন্ট কনফার্ম হলেই আমরা কাজ শুরু করি।

  1. পেমেন্ট করুন (bKash / Nagad / Rocket)
  2. আপনার GitHub অ্যাকাউন্টে 2FA চালু করুন
  3. আমাদের আপনার GitHub ইউজারনেম দিন
